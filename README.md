* [`handout.Rmd`](handout.Rmd): R Markdown source
* [`handout.pdf`](handout.pdf?raw=true): PDF copy
* [`handout-code.r`](handout-code.r): All R code used in the handout
* `accre/`: Sample R file and Bash script for running on ACCRE
